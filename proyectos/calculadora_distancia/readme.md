# DISTANCIA ENTRE PUNTOS EN EL PLANO

## Descripción
Script desarrollado en Python que calcula la distancia entre dos puntos en el plano cartesiano a partir de sus coordenadas. El programa solicita al usuario el ingreso de los valores, valida que los datos sean numéricos (float) y realiza el cálculo utilizando la fórmula de distancia euclidiana.

Este proyecto está orientado a la práctica de lógica de programación, validación de datos y modularización del código, aplicando conceptos matemáticos reales utilizados en agrimensura.

## Funcionalidades
- Ingreso interactivo de coordenadas (x, y)
- Validación de entradas numéricas (float)
- Cálculo de distancia euclidiana entre dos puntos
- Código modularizado mediante funciones
- Documentación interna utilizando docstrings

## Tecnologías utilizadas
- Python 3

## Objetivo del proyecto
Practicar el desarrollo de scripts en Python, enfocándose en:
- Uso de funciones
- Validación de datos
- Claridad del código
- Documentación
